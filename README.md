# ServiceNow GRC & IAM Project

## Overview
This project demonstrates an end-to-end implementation of IT Access Management GRC using ServiceNow IRM.

## Modules Covered
- Risk Management
- Control Management
- Control Testing
- Key Risk Indicators
- Issue & Remediation
- Audit Engagement

## Scenario
Unauthorized access risk in Finance Department.

## Implementation
1. Created risk register.
2. Designed access review control.
3. Built test plan and executed control test.
4. Defined KRI.
5. Raised issue and remediation task.
6. Conducted audit engagement.

## Outcome
Risk reduced from Assess to Monitor.
Control effectiveness improved.
Compliance achieved.

## Tools
- ServiceNow IRM
- GRC Workspace
